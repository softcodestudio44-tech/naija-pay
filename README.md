# NaijaPay 🇳🇬

Dead-simple Paystack integration for React. Accept payments in 5 lines of code.

## Why?

Traditional Paystack integration:
- 50+ lines of boilerplate
- Manual script loading
- Error handling nightmares
- Confusing documentation

NaijaPay:
- 1 import
- 1 component
- Built-in validation
- Automatic error handling

## Installation

```bash
npm install naija-pay
